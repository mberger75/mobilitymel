# MobilityMEL

### A website that allows you to see transport options in Lille

## Preview

![MobilityMEL Preview](screenshot.png 'MobilityMEL Preview')

## Stack

- React + TypeScript + Vite + sass
- Tanstack React-Query
- React-Leaflet

## How to run

Install packages

```bash
npm install
```

Run dev server

```bash
npm run dev
```
